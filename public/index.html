📖 How to Play

🎯 The Goal

A secret 4-color code has been hidden. Guess it within 10 tries to win!

🎨 The Colors

💡 How It Works

  • Click colors from the palette to build your 4-color guess.
  • Click a filled slot to remove that color.
  • Hit Submit Guess to lock it in.
  • After each guess you get feedback (see below).

📊 Reading the Feedback

Right spot — correct color AND correct position
Wrong spot — correct color but wrong position
Not in code — this color isn't in the secret at all

⚙️ Easy vs Hard Mode

  • 😌 Easy: The secret code has no repeated colors — each color appears at most once.
  • 🔥 Hard: Colors can repeat in the secret code!
